## Usage

[Just](https://just.systems/man/en/packages.html) and [Nix](https://nixos.org/download/) need to be installed.

See available commands with 
```bash
just
```

## Hosts/Outputs

* **liquid [nix-darwin]**: M1 Notebook that I use for most personal projects and workloads
* **meryl [NixOS]**: x86 Notebook for work
* **otacon [NixOS | Remote Deployment | SD Image]**: aarch64 SOC that runs Klipper for my 3D printer.
* **solid [Standalone Home Manager]**: x86 Not-NixOS host that i use for gaming.
* **mantis [NixOS | Remote Deployment]**: x86 Notebook mainboard that is being abused as a home server
